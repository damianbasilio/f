I've designed a bespoke, editorial-style homepage for Aloha Tanning that moves away from standard local business templates to establish a premium, "Sunset Noir" visual identity.

### Design Highlights:
- **Visual Identity:** The "Sunset Noir" palette uses deep charcoals and rich bronzes to evoke a high-fashion, golden-hour aesthetic.
- **Editorial Layout:** I implemented a high-contrast typographic system using *Playfair Display* for oversized headings, paired with thin architectural rule lines to create a structured yet sophisticated feel.
- **Immersive Imagery:** The design features custom-generated editorial photography—from the "Perpetual Glow" hero image to the "Science Meets Serenity" interior shots—to position Aloha Tanning as a boutique destination rather than a typical salon.
- **Functional Sophistication:** The layout replaces generic card grids with a custom, border-separated menu for services, ensuring the "The Aloha Standard" is felt in every pixel.
- **Pensacola Presence:** Local details are integrated throughout, including a custom visit section with a dedicated map slot and integrated contact form.

The resulting code is a production-ready, fully responsive semantic HTML document with custom CSS variables and smooth-scroll interactions.
